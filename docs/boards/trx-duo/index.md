# TRX Duo

TRX Duo - это дешевый аналог знаменитой платформы [Red Pitaya](https://redpitaya.com/)

У платформы есть даже [свой чат](https://t.me/+sxlqofO7-cw0ODJi) в Телеграмме.

![alt_text](https://www.sdrstore.eu/images/detailed/13/14256ea948.webp)

## О плате

Плата является не просто отладкой, но больше кастомной копией Red Pitaya, поэтому имеет не типичный набор пеериферии

### PL

- ADC LTC2208 x2
- DAC AD9767 x2
- GPIO

### PS

- USB type-C UART
- Ethernet
- USB type-C
- 512 MB DDR3
- USB 2.0
- microSD slot

## Питание

Плата работает от USB и не требует дополнительного внешнего блока питания.
